## Scope

MARADMIN 022/25 cancels MARADMIN 361/25, Fiscal Year 2026 Lieutenant Colonel Command Screening Board Results. The cancellation message issues from CMC Washington DC MRA MM. The original results message is no longer in effect. The MARADMIN is a short administrative message.

## Audience

- Marines on the FY26 Lieutenant Colonel Command Screening Board slate. The original results no longer stand. Watch for a follow-on results message.
- Manpower Management Officer Assignments (MMOA-3). Adjudicates command assignment questions post-cancellation.
- S-1 staff and adjutants. Confirm no actions reference the cancelled results.

## Action items

1. Confirm no command slate actions reference MARADMIN 361/25.
2. Subscribe to follow-on MARADMINs from MMOA-3 for the revised results.
3. Coordinate with MMOA-3 on individual cases.

## Connection to Marine Corps doctrine

- MARADMIN 361/25. Fiscal Year 2026 Lieutenant Colonel Command Screening Board Results. The cancelled prior message.
